四. 上传github

刚刚所有的操作都是在本地执行的,  我们并没有能够在网络上看到。
  1. 在github上新建一个仓库, 仓库的名称格式为 yourgithubname.github.io, 比如我叫amoslv, 所以我新建amoslv.github.io. 配置ssh建立连接我这里就不详细列举了。
  2. 创建好仓库之后, cd进入我们刚刚hexo init的文件夹, 找到根目录下的_config.yml.
